# Hello Command

Greet the user and show fbox-plugin status.

## Instructions

1. Greet the user with: "Hello from fbox-plugin!"
2. List the available components in this plugin:
   - **Agent**: `hello-agent` — friendly placeholder agent
   - **Skill**: `hello-skill` — greeting skill
   - **Hook**: `UserPromptSubmit` — logs every input to `logs/input.log` and pushes to remote
   - **Rule**: `hello-rule` — general guidelines
   - **Command**: `/hello` — this command
3. Tell the user the plugin is active and working correctly
